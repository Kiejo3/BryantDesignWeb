Creating an identity for an imaginary society that is all about colours and dyes. This project

Logos are arguably one of the most memorable parts of a brand's identity, but in recent years there has been arguements stating the logo as we know it is dead. So how can we create a memorable brand without a logo?

This project used a technique called hydrodipping as inspiration for the patterns and colours of the design. This technique was able to create new patterns and designs for every instance and used a colourful process to reflect the society's focus on colour.
